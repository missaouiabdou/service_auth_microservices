# Project Summary
This project is an Authentication Microservice that provides secure user registration, login, and token management functionalities. Built with Symfony 7 and PHP 8.4, it adheres to modern architectural principles such as Clean Architecture and SOLID principles. The service is containerized using Docker, utilizes PostgreSQL for data persistence, and integrates RabbitMQ for messaging, ensuring scalability and maintainability. The project is hosted on GitHub, making it accessible for collaboration and version control.

# Project Module Description
- **Infrastructure Layer**: 
  - Docker setup with Nginx, PostgreSQL, RabbitMQ, and Redis.
  - Configuration files for services and security.
  
- **Domain Layer**: 
  - Entities, value objects, and enums defining core business logic.
  
- **Application Layer**: 
  - Services for authentication, registration, token management, and event processing.
  - DTOs for data transfer.
  
- **Presentation Layer**: 
  - REST API controllers for handling requests and responses.
  - Custom exception handling.
  - Outbox Pattern, Circuit Breaker, and Rate Limiter for enhanced reliability.

# Technology Stack
- **Backend**: PHP 8.4, Symfony 7
- **Database**: PostgreSQL 16
- **Containerization**: Docker, Docker Compose
- **Message Broker**: RabbitMQ
- **Caching**: Redis
- **Logging**: Monolog
- **Authentication**: LexikJWTAuthenticationBundle

# Usage
1. **Install Dependencies**: 
   ```bash
   make composer-install
   ```
2. **Build and Start Services**: 
   ```bash
   make init
   ```
3. **Generate JWT Keys**: 
   ```bash
   make generate-keys
   ```
4. **Run Migrations**: 
   ```bash
   make migrate
   ```
5. **Process Outbox Events**: 
   ```bash
   make process-outbox
   ```
6. **View Logs**: 
   ```bash
   make logs
   ```
